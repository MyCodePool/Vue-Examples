# Vue Maps (Screenshot below)


## Preparation

### Setup Vue CLI
```
npm install -g @vue/cli
```

### Setup ESLINT
```
npm install --save-dev eslint eslint-plugin-vue
```


## Project 

### Setup
```
npm install
```

### Projekt Fix in case of errors (optional)
```
npm audit fix --force
```

### Run & Enjoy
```
npm run serve
```

### Google Map API (optional)
Add Google API KEY in main.js line 10
